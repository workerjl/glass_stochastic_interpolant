# glass_stochastic_interpolant

Sample from Sherrington Kirkpatrick spin glass model with normalizing flows trained using stochastic interpolant framework. Train using reverse KL (data-free) or forward KL (dataset built with parallel tempering Monte Carlo).

Code spliced from https://github.com/gshartnett/deepglass and https://github.com/malbergo.

Albergo, Boffi, Vanden-Eijnden. "Stochastic Interpolants: A Unifying Framework for Flows and Diffusions" https://arxiv.org/abs/2303.08797

Hartnett, Mohseni. "Self-Supervised Learning of Generative Spin-Glasses with Normalizing Flows" https://arxiv.org/abs/2001.00585
